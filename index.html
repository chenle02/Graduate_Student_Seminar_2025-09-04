Longest Increasing Subsequence

A key concept in understanding certain random processes is finding the longest increasing subsequence (LIS) within a sequence of numbers.

LIS Example

Let's find the Longest Increasing Subsequence for:

[0, 1, 0, 3, 2, 3]

Step 1: Initialization

dp: the length of the LIS ending at each index
prev: reconstruct the sequence

index 0 1 2 3 4 5
arr 0 1 0 3 2 3
dp 1 1 1 1 1 1
prev -1 -1 -1 -1 -1 -1

Step 2: DP Calculation

For each element arr[i], we look at all previous elements arr[j] (where j < i).

If these conditions are met:

  • arr[j] < arr[i]
  • dp[j] + 1 > dp[i]

...then we update:

  • dp[i] = dp[j] + 1
  • prev[i] = j

Let's see it in action...

DP Step: i = 1

index 0 1 2 3 4 5
arr 0 1 0 3 2 3
dp 1 2 1 1 1 1
prev -1 0 -1 -1 -1 -1

arr[0] < arr[1] (0 < 1), so dp[1] = dp[0] + 1 = 2. prev[1] = 0.

DP Step: i = 2

index 0 1 2 3 4 5
arr 0 1 0 3 2 3
dp 1 2 1 1 1 1
prev -1 0 -1 -1 -1 -1

No previous element is smaller than arr[2] (0). No change.

DP Step: i = 3

index 0 1 2 3 4 5
arr 0 1 0 3 2 3
dp 1 2 1 3 1 1
prev -1 0 -1 1 -1 -1

arr[1] < arr[3] (1 < 3) gives the best update: dp[3] = dp[1] + 1 = 3. prev[3] = 1.

DP Step: i = 4

index 0 1 2 3 4 5
arr 0 1 0 3 2 3
dp 1 2 1 3 3 1
prev -1 0 -1 1 1 -1

arr[1] < arr[4] (1 < 2) gives the best update: dp[4] = dp[1] + 1 = 3. prev[4] = 1.

DP Step: i = 5

index 0 1 2 3 4 5
arr 0 1 0 3 2 3
dp 1 2 1 3 3 4
prev -1 0 -1 1 1 4

arr[4] < arr[5] (2 < 3) gives the best update: dp[5] = dp[4] + 1 = 4. prev[5] = 4.

Step 3: Final DP Arrays

The LIS length is the maximum value in the dp array, which is 4.

index 0 1 2 3 4 5
arr 0 1 0 3 2 3
dp 1 2 1 3 3 4
prev -1 0 -1 1 1 4

Step 4: Reconstructing the Sequence

Start from the index of the max dp value (index 5) and trace back using prev.

  1. Start: index 5. Sequence: [3]
  2. prev[5] = 4. Prepend arr[4] (2). Sequence: [2, 3]
  3. prev[4] = 1. Prepend arr[1] (1). Sequence: [1, 2, 3]
  4. prev[1] = 0. Prepend arr[0] (0). Sequence: [0, 1, 2, 3]
  5. prev[0] = -1. Stop.

The Result

The Longest Increasing Subsequence is:


[0, 1, 2, 3]

Length: 4

LIS and Last Passage Percolation (LPP)

The LIS of a random permutation is equivalent to the last passage time in a directed percolation model via the Robinson–Schensted correspondence. These visuals connect LIS paths to maximal-weight up-right paths.

Permeation and Young Tableaux

The Robinson–Schensted–Knuth (RSK) correspondence maps a permutation to a pair of Young tableaux (P, Q) via row insertion. The LIS length equals the length of the first row of P, linking LIS to tableaux structure and, via a growth picture, to LPP.

The Tracy-Widom Distribution

A fascinating probability distribution arising in Random Matrix Theory.

What is it?

  • Describes the distribution of the normalized largest eigenvalue of a random Hermitian matrix.
  • Introduced by Craig Tracy and Harold Widom.

Where does it appear?

  • Random Matrix Theory: Its original context.
  • Longest Increasing Subsequence (LIS): The length of the LIS of random permutations follows a Tracy-Widom distribution.
  • Kardar-Parisi-Zhang (KPZ) Equation: Describes large-scale statistics in this universal growth model.

Significance

  • A universal distribution, appearing in diverse physical and mathematical systems.
  • Exhibits a third-order phase transition, analogous to statistical mechanics.
  • Defined via a Fredholm determinant, related to Painlevé transcendents.
